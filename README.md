# Loopholes in Bell Measurement

A personal research and learning repository containing **Mathematica notebooks** related to Bell measurement scenarios, loopholes in Bell-type tests, and associated analytical/numerical explorations.

## About

This repository collects my working notebooks on topics related to:

- Bell measurements
- loopholes in Bell-type experiments
- ensemble-based constructions
- Gisin-type scenarios
- thermal and singlet-state analyses
- unnormalised / alternative formulations explored during calculation

The goal of this repository is to keep my derivations, numerical experiments, and exploratory calculations organized in one place.
